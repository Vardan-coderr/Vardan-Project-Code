3. List Tag , Table Tag And Form Tag

* HTML List :-

. Lists are used to represent real life list data.
. HTML lists allow web developers to group a set of related items in lists.


* List are two types:-
. Unordered List
. Ordered List


* Unordered List :- An unordered list typically is a bulleted list of items.

* Ordered List :-
. An ordered list can be numerical or alphabetical.
. The li tag is used to define each list item.




* HTML Table :-

* What is a Table Tag ?

. An HTML structure for creating rows and columns on a Web page


* Table Headers :-

. Sometimes you want your cells to be table header cells. In those cases use the (th) .

* Table Row :-

. The (tr) HTML element defines a row of cells in a table.


* Table Data :-

A table cell or data can be added using the (td) tag.

*HTML Form :-

* What is a form in HTML?

. An HTML form is used to collect user input.

. Forms in HTML documents . An HTML form is a section of a document containing normal content, markup, special elements called controls (checkboxes, radio buttons, menus, etc.), and labels on those controls.
